Sync the team's Orgtology from Palette into PALETTE.md in the current directory.
Follow the instructions in the Palette Orgtology skill to fetch the orgtology://
resource (or a specific slice if the user specified one), write it to PALETTE.md,
ensure one project instruction file references PALETTE.md (`CLAUDE.md`,
`AGENTS.md`, `CURSORRULES`, or `.cursorrules`), creating `CLAUDE.md` with the
reference if none exists, and summarize what changed.
